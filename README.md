## Overview
**MaaNTE** is an advanced, third-party automation assistant designed to optimize your gameplay in *Neverness to Everness (NTE)*. It is fully powered by the highly efficient **MAAFramework**.
The software leverages intelligent image recognition to safely simulate regular user interactions, allowing you to completely eliminate tedious, repetitive daily tasks without shifting game balance or modifying any internal game files and memory.

> [!Important]
> This project is open-source, completely free, and intended exclusively for personal educational and convenience purposes.

## Key Features
- **Auto Fishing System:** Fully automated fishing process including auto-buy bait and auto-sell fish.
- **Cafe Management Helper:** Automated coffee making and automated cafe revenue extraction.
- **Dialogue Accelerator:** Auto-skip story dialogue and cutscenes to save time.
- **Daily Routine Automation:** Auto-claim daily rewards and log in tasks instantly.
- **Smart Image Recognition:** Driven by MAAFramework for precise simulation of mouse and keyboard inputs.
- **Safe Operations:** Strictly adheres to client-side interaction boundaries without memory injection.

## How to Use
1. Download the latest release archive from the releases section.
2. Extract the package into your desired project directory.
3. Launch *Neverness to Everness* and configure the recommended screen resolution.
4. Run `MaaNTE.exe` as Administrator.
5. Select your preferred automation profile (Fishing, Cafe, or Dialogue Skip) and start the task execution.
