## 🛠️ Key Features 

### Auto Fishing
The tool watches the screen for the fishing minigame. It identifies the bobber movement and provides clicks to catch fish. You no longer need to watch the screen for long periods.

### Dialogue Skipping
Modern games contain many conversations. MaaNTE detects when dialogue boxes appear and clicks the skip button. This allows you to finish quests faster.

### Daily Rewards
You lose resources if you forget to claim daily login rewards. The tool navigates to the calendar or shop menu every day to claim these items for your account.

## 🛡️ Software Safety 

MaaNTE runs as a local macro on your computer. It reads your screen pixels to make decisions. It does not modify game files. This approach keeps your game installation clean. We recommend using this tool only for personal account management.

## 📋 Troubleshooting Common Issues

### The tool does not click buttons
Ensure the game window is not covered by other windows. MaaNTE needs a clear view of the screen.

### The tool clicks the wrong spots
Check your game resolution. MaaNTE assumes a standard 16:9 aspect ratio. If you use a wide display, the coordinate mapping might fail. Change your game settings to 1920x1080 to fix this.

### The program fails to start
Check your virus protection software. Some antivirus tools block automation scripts by default. Add the MaaNTE folder to your exclusion list to allow the program permissions to function.

### The tool stops working after a game update
Game developers often change the game interface. This can break the screen recognition logic. Visit the GitHub page to see if a newer version exists. Updates provide fixes for these interface changes.
